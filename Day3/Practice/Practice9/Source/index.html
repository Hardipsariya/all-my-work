Var

The var statement is used to declare a variable in JavaScript. A variable declared with the var keyword is defined throughout the program.


It has global scope.


Variable declared with var keyword can be re-declared and updated in the same scope.

Let

The let statement is used to declare a local variable in TypeScript. It is similar to the var keyword, but it has some restriction in scoping in comparison of the var keyword.


It is limited to block scope.


Variable declared with let keyword can be updated but not re-declared.
